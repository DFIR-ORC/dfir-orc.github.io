Configuring process priority¶

To minimize tools’ impact on the analysed systems, all tools accept the /low command line option. This will set the process priority class to BELOW_NORMAL_PRIORITY_CLASS. The main thread priority is set to THREAD_PRIORITY_BELOW_NORMAL.
